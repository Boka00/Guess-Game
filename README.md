# Guess My Number 🎯

A simple number guessing game built with HTML, CSS, and JavaScript.
## Link 🔗 https://boka00.github.io/Guess-Game/
## How to Play

1. The game generates a secret number between **1 and 20**.
2. Enter your guess in the input field.
3. Click **Check!** to submit your guess.
4. The game will tell you whether your guess is:
   - Too high
   - Too low
   - Correct
5. Try to guess the number with the highest score possible.
6. Click **Again!** to start a new game.

## Features

- Random number generation
- Score tracking
- High score tracking
- Responsive UI
- Restart functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
